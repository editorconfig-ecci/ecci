# Agent Instructions

## Clarifying Material Ambiguities

Before proceeding, ask the user a concise, user-facing clarifying question when a
material ambiguity remains in the requirements, intended behavior, acceptance
criteria, or priority and no safe assumption can be made. State the decision the
user needs to make. Do not silently choose product behavior on the user's behalf.

Resolve routine, low-risk implementation details from established repository
conventions without unnecessary questions.

This file is agent-facing guidance. Keep human-facing documentation separate and
write all documentation in English.
